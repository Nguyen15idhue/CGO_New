# CHC Geomatics Office 2 - User Guide

## 1. Package Contents
This package only needs these files:
- CHC Geomatics Office 2.exe
- KeyGen.exe (admin only, do not send to end users)

For end users, usually only CHC Geomatics Office 2.exe is required.

## 2. What This App Does
- Runs CHC Geomatics Office 2 with USB-based protection.
- First run requires activation key.
- After successful activation, next runs open directly.
- If USB is unplugged while software is running, the app is forced to close and protection attempts to restore original state.

## 3. First-Time Activation (User)
1. Plug in USB.
2. Run CHC Geomatics Office 2.exe.
3. Activation window appears.
4. Copy Activation Code and send it to admin.
5. Enter Activation Key received from admin.
6. Click Confirm.

If key is valid, activation is saved and software starts.

## 4. Normal Daily Use
1. Plug in the same activated USB.
2. Run CHC Geomatics Office 2.exe.
3. Software opens directly (no activation window).

## 5. Important Rules
- License is bound to USB identity, not machine identity.
- Same USB can be used on different PCs.
- Copying files to another USB will require new activation.

## 6. If Activation Fails
- Check that you are running from removable USB.
- Re-check the key format: CHC-XXXX-XXXX-XXXX.
- Ensure key matches the current activation code.

## 7. If Software Closes After USB Unplug
This is expected behavior.
- Re-plug USB.
- Run CHC Geomatics Office 2.exe again.

## 8. Support Info to Send Admin
When reporting issues, send:
- Windows version
- What action caused issue (open/run/unplug/reopen)
- Screenshot of activation window or error popup
- Time of issue
